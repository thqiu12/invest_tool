# Quant Guard

本地量化交易与风控原型。它不会连接券商，也不会自动下单，主要用于：

- 上传价格 CSV 并回测策略
- 比较策略收益、最大回撤、胜率和交易次数
- 计算高杠杆 ETF 的仓位上限
- 估算再跌 30% 时对总资金池的影响
- 给出买入、卖出或观望信号
- 导入楽天证券或自定义持仓 CSV
- 监控 3 倍 ETF、科技/半导体集中度
- 计算 SOXL 回本、继续下跌、减仓后的风险变化

## 使用方法

直接用浏览器打开 `index.html` 即可。

CSV 至少需要两列：

```csv
Date,Close
2026-06-01,250.30
2026-06-02,243.10
```

## 持仓 CSV

左侧“持仓导入”可以上传楽天证券的持仓 CSV，也可以上传自己整理的 CSV。推荐自定义格式：

```csv
Symbol,Name,Quantity,AvgCost,CurrentPrice,MarketValue,Pnl
SOXL,Direxion Daily Semiconductor Bull 3X,250,243.1092,182.13,6602213,-2210496
TECL,Direxion Daily Technology Bull 3X,60,251.9466,202.28,1760000,-432000
MRVL,Marvell Technology,20,299.02,263.47,764063,-103000
```

楽天 CSV 如果是 Shift-JIS 编码，页面会自动尝试读取。导入后系统会自动识别常见 3 倍 ETF，例如 SOXL、SOXS、TECL、TQQQ、SQQQ，并把 SOXL 带入修复计算器。

如果有完整行情，推荐格式：

```csv
Date,Open,High,Low,Close,Volume
2026-06-01,250.00,258.00,246.50,253.20,1000000
```

## 已有策略

- 均线交叉：快线向上穿慢线买入，向下穿卖出
- RSI 反转：RSI 低于下限买入，高于上限卖出
- 20 日突破：突破近期高点买入，跌破近期低点卖出

## 风控原则

这个工具默认把 SOXL、TECL、TQQQ、SOXS 这类每日 3 倍 ETF 当成高风险交易仓。建议先控制：

- 3 倍 ETF 总仓位占总资金池比例
- 单笔亏损容忍
- 等效方向暴露
- 再跌 30% 的绝对损失

资产驾驶舱会显示：

- 导入持仓总市值
- 导入持仓盈亏
- 3 倍 ETF 占总资产比例
- 科技/半导体占总资产比例
- 每个持仓的风险状态

SOXL 修复计算器会显示：

- 回到平均成本需要上涨多少
- 再跌 10%、20%、30% 的损失
- 反弹 20% 的影响
- 减仓到目标上限、减仓 50%、减仓 75% 后的风险

所有输出只用于研究和记录，不构成投资建议。
